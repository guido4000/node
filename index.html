How to setup a VPS node webserver in 3 minutes

Spin up an EC2 instance, install node, run server, get HTML5 boilerplate - done!

1. Get the server

Free 30 minute one click server at Instantserver.io
For a more permanent solution check out DigitalOcean.com

2. Install node

$ sudo apt-get update
$ sudo apt-get install python-software-properties python g++ make
optional $ sudo apt-get install software-properties-common
$ sudo add-apt-repository ppa:chris-lea/node.js
$ sudo apt-get update
$ sudo apt-get install nodejs

3. Install connect

a) $ npm install connect

b) create server.js

var connect = require('connect');
connect.createServer(
connect.static(__dirname)
).listen(8080);

c) run server
$ node server.js

4. Get HTML5 Boilerplate

HTML5 Boilerplate
Extract and copy all files to the server

5. Open website in Browser on port 8080

Done!
